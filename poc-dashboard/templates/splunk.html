Splunk — Security Analytics

Centralised security logging — receives events from Cisco Duo, Identity Intelligence, and Cisco Secure Access, and exposes search to Claude via the Splunk MCP Server.

WHAT IS SPLUNK?

Splunk is the security analytics platform in this PoC. It ingests structured log data from Cisco Duo MFA, Cisco Identity Intelligence, and Cisco Secure Access — giving you a single place to search, correlate, and visualise authentication events, policy decisions, and network activity.

The Splunk MCP Server running alongside it exposes Splunk search to Claude and other AI agents, enabling natural-language queries over your security logs — for example: "show me all failed Duo authentications in the last hour" or "which users accessed the cluster today via Secure Access?"

CONFIGURATION STEPS IN CISCO IDENTITY INTELLIGENCE
  1. Navigate to Integrations in the CII portal.
  2. Under Notification Targets, click Create New Target and select Webhook.
  3. Paste the Webhook URL above. Set the Authorization header to the value above. Use test_splunk_demo as the input name.
  4. Click Save, then click the ⋯ menu next to the integration and select Test Connectivity. A Success popup confirms the connection.
  5. In Splunk, go to App Analytics → Cisco Identity Intelligence Dashboard and confirm the test event appears.
