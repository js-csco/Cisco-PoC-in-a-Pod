ZTA PROFILE

☑ To do: Adjust the Default ZTA Profile to steer all internet traffic through Secure Access.
  1. In the Secure Access dashboard, go to Connect → End User Connectivity
  2. Find the Default ZTA Profile and click the pencil icon to edit it
  3. Open the Secure Internet Access tab
  4. Under Traffic Steering, select "Use ZTA to secure all internet destinations"
  5. Save the profile

Security Profile

☑ To do: Enable HTTPS Decryption on the existing system profile and add it to all Internet Access Policies.
  1. Go to Secure → Security Profiles
  2. Click Edit on the System Provided - Internet Access profile
  3. Enable the Enable Decryption setting and save
  4. Add this profile to each of the Internet Access Policies created above

DATA LOSS PREVENTION

Creates two DLP rules: AI Guardrails block sensitive prompts sent to AI applications, and Real-Time DLP blocks uploads or pastes of PCI and PII data across all proxied web traffic.

Rule type What it blocks Where Test examples
