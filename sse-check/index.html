HOW ACCESS IS DETECTED

The Resource Connector runs on this server as a pod with IP SERVER_IP. When you access via Cisco Secure Access, your traffic travels through the Secure Access cloud and exits through the on-premises connector — so this application only ever sees SERVER_IP as the source, never your real IP. Accessing directly over the LAN or internet exposes your actual IP instead.
