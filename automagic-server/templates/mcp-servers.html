Connect Claude Desktop to this PoC environment via the Model Context Protocol (MCP). MCP lets Claude query your cluster directly — no copy-pasting logs or screenshots.

WHAT IS MCP?

The Model Context Protocol is an open standard that lets AI models like Claude call tools on external servers in real time. Instead of pasting logs into Claude, you configure Claude Desktop with the server URL below — Claude then queries your cluster live during the conversation.
