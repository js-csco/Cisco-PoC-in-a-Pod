ZTA PROFILE

☑ To do: Adjust the Default ZTA Profile to steer all internet traffic through Secure Access.
  1. In the Secure Access dashboard, go to Connect → End User Connectivity
  2. Find the Default ZTA Profile and click the pencil icon to edit it
  3. Open the Secure Internet Access tab
  4. Under Traffic Steering, select "Use ZTA to secure all internet destinations"
  5. Save the profile
