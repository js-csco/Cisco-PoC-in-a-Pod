Secure Access

Cisco Secure Access is a converged SSE platform that unifies Zero Trust Network Access (ZTNA), Secure Web Gateway (SWG), CASB, DNS-layer security, and DLP into a single cloud-delivered service.

Use the sections below to automate the initial configuration of your PoC environment directly via the Cisco Secure Access APIs.

ZTA PROFILE

☑ To do: Adjust the Default ZTA Profile to steer all internet traffic through Secure Access.
  1. In the Secure Access dashboard, go to Connect → End User Connectivity
  2. Find the Default ZTA Profile and click the pencil icon to edit it
  3. Open the Secure Internet Access tab
  4. Under Traffic Steering, select "Use ZTA to secure all internet destinations"
  5. Save the profile
