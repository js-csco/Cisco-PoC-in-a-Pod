Quick access to all services running in the PoC environment.

CISCO CLOUD SERVICES

Cisco Secure Access
Cloud · ZTNA Gateway

Provides Zero Trust Network Access. The on-prem Connector in the cluster tunnels traffic from the cloud gateway to private resources — no inbound firewall rules needed.
Duo
Cloud · Identity & MFA

Multi-factor authentication and Identity Intelligence. Provides passwordless MFA, device trust verification, and risk-based authentication policies integrated with Secure Access.
ThousandEyes
Cloud · Network Monitoring

End-to-end network path visibility. Monitors connectivity and performance for the ZTNA access path from user to application.

CLUSTER INFRASTRUCTURE

Cilium
k3s DaemonSet · CNI / Network Policy

eBPF-based CNI providing Layer 4/7 network policies and microsegmentation. Enforces which pods can communicate and on which ports — without iptables overhead.
Tetragon
k3s DaemonSet · Runtime Security

eBPF-based runtime security. Monitors process execution, file access, network calls, and syscall activity across all pods in real time. Events are exported to Splunk.
Splunk
Pod · Port 30500 · SIEM

Log aggregation and SIEM. Ingests security events from Tetragon, Cilium, and application logs for full visibility, dashboards, and alerting.

DEMO APPLICATIONS & TOOLS

SSE Check
Pod · Port 30550 · Demo App

A demo web application serving as the private resource behind Zero Trust. Accessible only through Cisco Secure Access policies — not directly from the internet.

RDP / SSH Servers
Pods · Port 30389 / 30022

Remote desktop and SSH servers exposed as private resources. Demonstrates Cisco Secure Access protecting non-web protocols via browser-based agentless access.

PoC Playbook
Pod · Port 30250 · Playbook

Guided playbook for validating the full PoC — covering Duo, Secure Access, Cilium, and Tetragon with pass/fail tracking.

Uptime Kuma
Pod · Port 30300 · Status Monitoring

Uptime monitoring dashboard. Tracks availability and response times for all services in the cluster with pre-configured health checks.

Caldera C2
Pod · Port 30888 · Red Team

MITRE Caldera automated adversary emulation. Runs attack scenarios against the cluster for validating detection and enforcement.
MCP Servers
Pod · Port 30080 · AI Tooling

Model Context Protocol servers providing kubectl and cluster management capabilities to AI assistants.
