Cisco Duo

Cisco Duo is a multi-factor authentication (MFA) and Zero Trust access platform that verifies user identities and device health before granting access to applications and resources.

Use the sections below to connect to your Duo Admin API, create a test user, and set up integrations for your PoC environment.

GET DUO

Don't have a Duo account yet? Sign up for a free trial to get started.

Sign Up for Duo

AUTHENTICATION

Connect to Cisco Duo Admin API with your API hostname, integration key, and secret key.

USER & GROUP CREATION

Create a test user for your Duo PoC environment. Click "Setup Duo" to create the user, the "PoC Users" group, and assign the user to the group.

Add User

Setup Duo will perform the following:
  • Create user with email and username
  • Create "PoC Users" group
  • Add user to "PoC Users" group
  • Send enrollment email to user

INTEGRATIONS

A single integration connects Duo with Cisco Secure Access, covering both SSO and user provisioning (SCIM). In the Duo Admin Panel, go to Applications → Protect an Application, search for Cisco Secure Access, and follow the setup guide to configure SSO and enable user provisioning.
