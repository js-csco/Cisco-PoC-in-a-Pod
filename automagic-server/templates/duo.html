Cisco Duo

INTEGRATIONS

Two integrations are required to connect Duo with Cisco Secure Access. These must be configured manually in both dashboards.

1 User Directory — sync Duo users into Secure Access

This integration provisions Duo users into Cisco Secure Access so they can be assigned to policies.

  1. In the Duo Admin Panel, go to Users → Directory Sync and add a new directory.
  2. Choose Cisco Secure Access as the target and follow the SCIM setup wizard.
  3. In the Secure Access Dashboard, go to Users & Groups → User Management and confirm users appear after the first sync.
2 SAML Identity Provider — set Duo as the IdP for Secure Access

This integration makes Duo the authentication provider so users are challenged with Duo MFA when accessing resources through Secure Access.

  1. In the Duo Admin Panel, go to Applications → Protect an Application and search for Cisco Secure Access.
  2. Copy the Entity ID, SSO URL, and download the Certificate.
  3. In the Secure Access Dashboard, go to Settings → Identity → Identity Providers and add a new SAML provider using the values from Duo.
  4. Copy the SP Entity ID and ACS URL from Secure Access back into the Duo application settings and save.
