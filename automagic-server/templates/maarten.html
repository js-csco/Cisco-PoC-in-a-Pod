ABOUT CISCO

Cisco Systems is a worldwide technology leader specialising in networking, cybersecurity, cloud, and collaboration solutions. Founded in 1984 and headquartered in San Jose, California, Cisco connects organisations of all sizes — from small businesses to global enterprises and governments — enabling them to securely digitise operations and achieve their strategic goals.

With a portfolio spanning hardware, software, and services, Cisco is consistently recognised as one of the world's most innovative and trusted technology companies.

CISCO SECURITY PORTFOLIO

Cisco's security portfolio is one of the most comprehensive in the industry, covering every layer of the modern attack surface — from the network edge to the cloud workload.

Cisco Secure Access (SSE)

A cloud-delivered Security Service Edge platform that converges Zero Trust Network Access (ZTNA), Secure Web Gateway (SWG), Cloud Access Security Broker (CASB), Firewall-as-a-Service (FWaaS), and DNS-layer security into a single unified service. It ensures users can access any application, from anywhere, securely.

Cisco Duo

A leading Multi-Factor Authentication (MFA) and Zero Trust access solution. Duo verifies the identity of users and the health of their devices before granting access to applications, protecting organisations against credential theft and account takeover attacks.

Cilium & Tetragon

Open-source, eBPF-based projects under the CNCF umbrella. Cilium provides high-performance networking, observability, and network policy enforcement for Kubernetes workloads. Tetragon delivers deep runtime security, detecting and blocking threats at the kernel level using eBPF tracing policies.

Splunk

Acquired by Cisco in 2024, Splunk is the industry-leading platform for security information and event management (SIEM), observability, and IT operations. It enables teams to investigate threats, monitor infrastructure, and respond to incidents at machine speed.

THIS PROOF OF CONCEPT

PoC in a Pod bundles a representative slice of the Cisco security stack into a self-contained Kubernetes environment that can be spun up in minutes. It is designed to let you experience the integration between these products hands-on, without needing a full production deployment.

  • Zero Trust access control via Cisco Secure Access
  • Multi-Factor Authentication via Cisco Duo
  • Kubernetes network policy enforcement via Cilium
  • Runtime threat detection via Tetragon
  • Centralised log aggregation via Splunk
  • AI-powered cluster inspection via Kubectl MCP
